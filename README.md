## ArraySort :arrows_counterclockwise:

**Orientação do Projeto**

Produza os programas que implemente os algoritmos de ordenação *SelectionSort*, *BubbleSort*, *InsertionSort*, *QuickSort* e *MergeSort* para 4 (quatro) arrays (vetores) contendo 100, 1.000, 10.000 e 100.000 números aleatórios, respectivamente. Cada programa deverá ser executado 10 vezes para cada entrada e obtido a média de tempo de execução.
